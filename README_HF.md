# 🌙 Momentos

Red social donde:
- 🕐 Solo puedes publicar **1 vez cada hora**
- ⏳ Los posts duran **24 horas**
- 💾 La comunidad puede hacer posts **permanentes** (10+ saves)

## Código

Disponible en GitHub: https://github.com/nxtry-of/Momentos

## Uso

1. Regístrate con un usuario
2. Publica tu primer momento
3. Espera 1 hora para el siguiente
4. ¡Salva los momentos que te gusten!
